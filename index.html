ตั้งค่าการเชื่อมต่อ Google Sheets
ทำครั้งเดียว — ทุกคนในทีมจะใช้ไฟล์ HTML นี้ร่วมกัน ข้อมูลจะซิงค์ผ่าน Google Sheets กลาง
1สร้าง Google Sheets ใหม่
ไปที่ sheets.google.com → สร้าง Spreadsheet ใหม่
ตั้งชื่อ เช่น แผนการปฏิบัติงาน RD
คัดลอก Spreadsheet ID จาก URL:
docs.google.com/spreadsheets/d/[ID อยู่ตรงนี้]/edit
2เปิด Apps Script และวาง Code
ใน Sheets → เมนู Extensions → Apps Script
ลบ code เดิม แล้ววาง code ด้านล่างนี้ทั้งหมด จากนั้น Deploy เป็น Web App
(Execute as: Me, Who has access: Anyone)
คัดลอก Web App URL ที่ได้มา
📋 คลิกดู Apps Script Code
3ใส่ข้อมูลด้านล่าง
ข้อมูลจะถูกบันทึกในเบราว์เซอร์ของเครื่องนี้
